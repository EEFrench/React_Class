Take Home Challenge

  • Create a new context called AuthContext which provides the following values:
    • The logged in user (initial value should be null).
    • A login function which takes a username and password as input and sets the username as the logged in user if the username and password are correct. (In a real app, you would want to validate the login credentials in a backend, but for now it's ok to just hardcode some credentials in your frontend app).
    • A logout function which takes no inputs and sets the logged in user to null.
  • In the LoginForm component, uses the login function provided by the AuthContext to sign-in a user when the form is submitted.
  • Use the AuthContext inside the App component to get the logged in user.
  • Using conditional rendering:
    • render the LoginForm is there is no logged in user
